# Telegram Intake Conventions

## Role of Telegram in this system

Telegram is not only an input channel. It is also the first review surface.

## Operational rules

- collect requests, corrections, links, and images in Telegram
- return preview packages to Telegram before publish
- use Telegram review as the first human checkpoint
- do not treat repo write as the first draft preview mechanism

## Preview package recommendation

- preview image or screenshot
- markdown draft attachment
- short summary text
- asset summary when relevant
